# TimeNexus
A prototype of a videogame where you can manipulate time!

## Controls
`WASD`: Movement  
`Shift`: Sprint  
`E`: Interact  
`P`: Pause  
`Left mouse button`: Clicking on buttons  
`Mouse wheel`: Scrolling through time
